Cadastro de Cliente

Ao acessar a janela de Cadastro de Cliente permiti inserir um novo Cliente na base da empresa. Podendo esse cliente realizar Aluguel de um ou mais equipamento de informática.

Acesso a janela de Cadastro de Cliente

  1. Acessa o menu superior da janela principal

  2. Busque no sub-menu de "Cadastro"

  3. Clique no item de "Novo Cliente" ;ou

  4. Clique o ícone "Novo Cliente" da janela principal

Cadastrar Cliente

  1. Informe corretamente o CPF no campo indicado e nome do cliente na parte superior da janela

  2. Preencha os dados na aba de "Dados Pessoais", inserindo corretamente a Data de Nascimento

  3. Insira os dados referente ao Endereço do cliente na aba "Endereço Completo"

  4. Pressione o botão "Cadastrar"

Alterar Dados do Cliente

  1. Consulte o Cliente a ser alterado através da janela de Consulta

  2. Informe os dados a serem alterados, com excessão do CPF

  3. Lembrando de não deixar nenhum campo vazio

  4. Pressione o botão "Atualizar"

  5. Confirme a alteração dos dados, após a janela será encerrada

Observação

  1. Caso o CPF esteja errado aparecerá uma mensagem de erro

  2. Caso o CPF informado já esteja na base retorna mensagem de aviso

  3. Caso algo campo não esteja preenchido, informará para o preenchimento do mesmo

  4. Ao pressionar o botão de "Cancelar" limpa os dados do formulário

  5. Ao pressionar o botão de "Sair" encerra o formulário de "Cadastro de Cliente"
