Cadastro de Equipamento

Ao acessar a janela de Cadastrado de Equipamento permitir a inserção de novos produtos no estoque. Os produtos podem ser classificados por categoria: Desktop, Impressora, Monitor, Notebook, Projetor e Tablet.

Acesso a janela de Cadastro de Equipamento

  1. Acessa o menu superior da janela principal

  2. Busque no sub-menu de "Cadastro"

  3. Clique no item de "Novo Equipamento" ;ou

  4. Clique o ícone "Novo Equipamento" da janela principal

Cadastrar Equipamento

  1. Preencha os campos corretamente na janela de cadastro

  2. Selecione na "Categoria" qual tipo de Equipamento está sendo cadastrado

  3. Pressione o botão "Seguinte"

  4. Será aberta a próxima janela do Equipamento da Categoria selecionada

  5. Preencha os campos nessa janela

  6. Pressione o botão "Cadastrar"

Alterar Dados do Equipamento

  1. Consulte o Equipamento a ser alterado através da janela de Consulta

  2. Informe os campos a serem alterados, com excessão do "Codigo" e "Categoria"

  3. Pressione o botão "Seguinte"

  4. Informe os campos a serem alterados

  5. Lembrando de não deixar nenhum campo vazio

  6. Pressione o botão "Atualizar"

  7. Confirme a alteração dos dados, após a janela será encerrada

Observação

  1. Caso algo campo não esteja preenchido, informará para o preenchimento do mesmo

  2. Ao pressionar o botão de "Cancelar" limpa os dados do formulário

  3. Ao pressionar o botão de "Sair" encerra o formulário de "Cadastro de Equipamento"
