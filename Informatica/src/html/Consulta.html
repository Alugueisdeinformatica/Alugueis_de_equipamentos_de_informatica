LPWS 2016-2 Exercício

1 - O que é Internet?

É uma plataforma de interação digital, na qual vários dados e informações são trocados/compartilhados atráves da rede(WAN).

2 - O que é e quais são as funções de HTML, CSS e JavaScript?

HTML é uma linguaguem de formatação da estrutura do documento.

CSS (Cascading Style Sheets),em português Folha de Estilo em Cascata, é utilizado pra definir aparência em páginas da internet.

JavaScript é uma linguaguem de programação interpretada, sendo usada para controlar o HTML e CSS, na manipulação de sub-itens.

3 - O que é um navegador para Internet?

É um software que permite seus usuários interagirem com documentos virtuais da Internet, conhecidas como páginas da web.

4 - O que é um plugin para navegador?

É um programa instalado no navegador que permite adicionar recursos ausentes na linguaguem HTML.
