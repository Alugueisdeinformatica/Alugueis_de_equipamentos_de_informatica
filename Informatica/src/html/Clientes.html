Consulta de Clientes

Ao acessar a janela de "Consulta de Cliente" terá acesso às informações dos clientes cadastrados. Podendo escolher a opção de alterar dados do cliente, sendo direcionado à janela de "Cadastro de Cliente", e na mesma atualizar os dados do cliente.

Acesso à janela de Consulta de Cliente

  1. Acesse o menu superior da janela principal;

  2. Busque no sub-menu de "Consulta";

  3. Clique no item de "Clientes".

Pesquisar Cliente para edição

  1. Informe o CPF do cliente no campo indicado ou selecione a opção de "Buscar Todos";

  2. Pressione o botão de "Buscar";

  3. Selecione na tabela o cliente desejado;

  4. Pressione o botão "Editar";

  5. Abrirá a janela para "Alterar os dados".

Pesquisar Cliente para inserir no Aluguel ou Efetuar Devolução

  1. Informe o CPF do cliente no campo indicado;

  2. Pressione o botão de "Buscar";

  3. Selecione na tabela o cliente desejado;

  4. Pressione o botão "Selecionar";

  5. Abrirá a janela para "Aluguel" ou janela para "Efetuar Devolução".

Exibir histórico de Aluguéis do Cliente

  1. Informe o CPF do cliente no campo indicado ou selecione a opção de "Buscar Todos";

  2. Pressione o botão de "Buscar";

  3. Selecione na tabela o cliente desejado;

  4. Pressione o botão "Histórico";

  5. Abrirá uma nova Janela com o demonstrativo dos aluguéis feitos pelo cliente.

Observação

  1. Caso o CPF esteja errado, aparecerá uma mensagem de erro;

  2. Caso ainda não tenha sido realizado o cadastro de cliente e realizada uma busca, retorna mensagem que a base de Cliente está vazia;

  3. Caso não preenchido o campo do CPF e realizada uma busca, informará para o preenchimento do mesmo.
