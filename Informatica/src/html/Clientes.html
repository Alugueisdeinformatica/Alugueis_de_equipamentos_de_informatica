Consulta de Clientes

Ao acessar a janela de consulta de cliente terá acesso as informações dos clientes cadastrados. Podendo escolher a opção de alterar dados do cliente, sendo direcionado a janela de "Novo Cliente", e na mesma atualizar os dados do cliente.

Acesso a janela de Consulta de Cliente

  1. Acessa o menu superior da janela principal

  2. Busque no sub-menu de "Consulta"

  3. Clique no item de "Clientes"

Pesquisar Cliente para edição

  1. Informe o CPF do cliente no campo indicado ou selecione a opção de "Buscar Todos"

  2. Pressione o botão de "Buscar"

  3. Selecione na tabela o cliente desejado

  4. Pressione o botão "Editar"
