1 - O que é Internet?

É uma plataforma de interação digital, na qual vários dados e informações são trocados/compartilhados através da rede (WAN).

2 - O que é e quais são as funções de HTML, CSS e JavaScript?

HTML é uma linguagem de formatação da estrutura do documento.

CSS (Cascading Style Sheets), em português Folha de Estilo em Cascata, é utilizado pra definir aparência em páginas da internet.

JavaScript é uma linguagem de programação interpretada, sendo usada para controlar o HTML e CSS, na manipulação de subitens.

3 - O que é um navegador para Internet?

É um software que permite seus usuários interagirem com documentos virtuais da Internet, conhecidas como páginas da web.

4 - O que é um plugin para navegador?

É um programa instalado no navegador que permite adicionar recursos ausentes na linguagem HTML.

5 - O que é uma extensão para navegador?

As extensões permitem adicionar novos recursos ao navegador, que podem modificar o conteúdo das paginas, fazer integração a outros serviços.

6 - Como representar um endereço de uma página na Internet?

Através de URL's, que são endereços para acessar uma página na internet ou a um aplicativo da web específico.

7 - O que é IP e DNS?

IP é uma série de números que informa onde se encontra um dispositivo na internet, seja um servidor ou computador pessoal.

DNS são responsáveis por localizar e traduzir para números IP os endereços dos sites que são digitados nos navegadores, também chamadas de URL's.

8 - O que é phishing?

O termo vem do inglês, semelhante a \"pescaria\". Trata-se de um site falso, no qual leva o usuário a atualizar suas informações. Geralmente são páginas semelhantes às de bancos, porém os dados inseridos são em um site impostor.

9 - O que é software livre e software proprietário?

Software livre é um programa criado e gerenciado por várias comunidades de desenvolvedores espalhadas pelo mundo. Estas comunidades se comprometem a colaborar com o desenvolvimento, atualização e correção do software.

Software proprietário é um produto criado por uma empresa que detém os direitos sobre o produto, e sua distribuição é feita mediante pagamento de licença de uso.

10 - Quais opções de aplicativos e serviços para troca de arquivos (pequenos e grandes) na Internet você já usou?
