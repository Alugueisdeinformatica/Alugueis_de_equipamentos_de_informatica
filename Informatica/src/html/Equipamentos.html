Consulta de Equipamentos

Ao acessar a janela de consulta de equipamento terá acesso as informações dos equipamentos cadastrados. Podendo escolher a opção de alterar dados do equipamento, sendo direcionado a janela de "Cadastro de Equipamento", e na mesma atualizar os dados do equipamento.

Acesso a janela de Consulta de Equipamento

  1. Acesse o menu superior da janela principal

  2. Busque no sub-menu de "Consulta"

  3. Clique no item de "Equipamentos"

Pesquisar Equipamento para edição

  1. Informe o código do equipamento no campo indicado ou selecione a opção de "Buscar Todos"

  2. Pressione o botão de "Buscar"

  3. Selecione na tabela o equipamento desejado

  4. Pressione o botão "Editar"

  5. Abrirá a janela para "Alterar os dados"

Observação

  1. Caso o código esteja errado aparecerá uma mensagem de erro

  2. Caso ainda não sido realizado cadastro de equipamento e realizada uma busca, retorna mensagem que a base de Equipamento está vazia

  3. Caso não preenchido o campo do código e realizada uma busca, informará para o preenchimento do mesmo
