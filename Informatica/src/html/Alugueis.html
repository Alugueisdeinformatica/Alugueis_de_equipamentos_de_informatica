Consulta de Aluguéis

Ao acessar a janela de consulta de alguel terá acesso as informações dos aluguéis em aberto. Ao selecionar algum registro da tabela será indicado se está dentro do prazo ou se o aluguel se encontra com atraso.

Acesso a janela de Consulta de Aluguel

  1. Acesse o menu superior da janela principal

  2. Busque no sub-menu de "Consulta"

  3. Clique no item de "Alugueis"

  4. Será apresentado todos os Aluguéis que estão em Aberto

Efetuar Devolução

  1. Clique na tabela no aluguel desejado para devolução

  2. Habilitará o botão de "Efetuar Devolução"

  3. Clique no botão "Efetuar Devolução"

  4. Abrirá a janela para "Aluguel"
