Efetuar Aluguel

Ao acessar a janela de "Novo Aluguel"

Acesso a janela de Novo Aluguel

  1. Acesse o menu superior da janela principal

  2. Busque no menu de "Aluguel"

  3. Clique no sub-menu de "Novo Aluguel" ;ou

  4. Clique o ícone "Novo Aluguel" da janela principal

Efetuar Aluguel

  1. Clique no botão "Inserir"

  2. Consulte o Cliente a ser inserido através da janela de Consulta

  3. Será habilitado o botão de "Buscar" e as opções de "Categoria"

  4. Selecione a categoria de Equipamento

  5. Clique no botão "Buscar"

  6. Selecione o Equipamento desejado

  7. Clique no botão "Selecionar"

  8. Informe a quantidade a ser alugada

  9. Será apresentado os detalhes do pedido na tabela ao lado "Pedido do Cliente"

  10. Caso deseje selecionar mais equipamentos de outra Categoria volte ao passo 4

  11. Para selecionar a mesma categoria volte ao passo 6

  12. Caso desejado remover algum item do pedido, selecione o item

  13. Clique no botão "Remover Item"

  14. Informe a quantidade de dias no campo indicado

  15. Clique no botão "Fechar Pedido" para efetuar o Aluguel

  16. Confirme o aluguel dos equipamentos

  17. Será habilitado o botão "Nova Venda"

  18. Clique no botão "Nova Venda" caso deseje fazer novo aluguel repetindo os passos acima

Observação

  1. Caso na busca do cliente não o encontre, então primeiramente cadastre um Novo Cliente

  2. Para realizar o aluguel certifique que tenha cliente cadastrado e equipamentos inseridos na base

  3. Certifique de informar a quantidade de dias para realizar o fechamento do pedido
